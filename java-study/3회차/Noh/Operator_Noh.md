# 연산자 (Operator)
## 연산자
* 연산을 수행하는 기호
### 연산자와 피연산자
* 연산자가 연산을 수행하려면  반드시 연산의 대상이 있어야 함 -> 피연산자
```java
연산자(operator) : 연산을 수행하는 기호(+, -, *, / 등)
피연산자(operand) : 연산자의 작업 대상(변수, 상수, 리터럴, 수식) 
```
```java
x + 3
// '+'는 두 피연산자를 더해서 그 결과를 반환하는 덧셈 연산자
// 변수 x와 상수 3은 이 연산자의 피연산자
```
